https://www.python.org/downloads/release/python-370/

Install xmodem package (untar first):

from dir:

..\tools\xmodem\Addon_Python_Packages\xmodem-0.4.4\

python setup.py install

Install pyserial package (untar first):

from dir:

..\tools\xmodem\Addon_Python_Packages\pyserial-3.0.1\

python setup.py install
